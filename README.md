# DevOps Portfolio Dashboard

## Overview

This project is a responsive personal portfolio website built using HTML, CSS, and JavaScript. It is designed to showcase my skills, projects, and learning journey in AWS, Linux, Cloud Computing, and DevOps.

The website serves as a professional portfolio where recruiters and hiring managers can view my technical skills and project experience.

---

## Features

* Responsive design for desktop and mobile devices
* Professional portfolio layout
* About Me section
* Technical Skills section
* Project showcase section
* Contact section
* Interactive JavaScript components
* Clean and modern user interface

## Installation and Setup

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/devops-portfolio-dashboard.git
```

### Navigate to Project Directory

```bash
cd devops-portfolio-dashboard
```

### Open in VS Code

```bash
code .
```

### Run Project

Open `index.html` in your browser or use the Live Server extension in VS Code.
